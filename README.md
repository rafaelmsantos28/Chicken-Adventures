## 🎮 Sobre o Projeto

O **Chicken Adventure** foi construído do utilizando a engine Unity e a linguagem C#. O foco do projeto foi aplicar os fundamentos completos do desenvolvimento de jogos, passando por todas as etapas: desde a configuração inicial do ambiente e mecânicas de física, até o level design, sonorização e publicação final (build).

## 🛠️ Tecnologias Utilizadas

* **Engine:** Unity
* **Linguagem:** C#

## ✨ Funcionalidades e Aprendizados

Durante a construção do jogo, os seguintes sistemas e conceitos foram implementados:

* **Controlador do Herói:** Programação da física de movimentação do personagem, incluindo mecânicas de pulo e ataque.
* **Sistema de Animação:** Gerenciamento da árvore de animações (Animator) para transições suaves entre estados (Idle, Correr, Pular, Atacar).
* **Inimigos e Coletáveis:** Criação de comportamentos para adversários, lógica de contato/dano e itens coletáveis para missões.
* **Interface de Usuário (UI):** Desenvolvimento de HUD para feedback visual ao jogador.
* **Construção de Mundo (Level Design):** Utilização da ferramenta *Tilemap* da Unity para montar os cenários e configuração das matrizes de colisão.
* **Áudio:** Implementação de trilha sonora de fundo e efeitos sonoros (SFX) vinculados às ações do jogo.
* **Build e Deploy:** Processo completo de compilação do projeto para a exportação e publicação do jogo finalizado.

## 🚀 Executando o Projeto

1. Certifique-se de ter a **Unity** instalada (versão 6.3 LTS ou superior).
2. Clone este repositório:
```bash
git clone https://github.com/rafaelmsantos28/Chicken-Adventure
```

4.  Abra o Unity Hub e selecione a pasta clonada.
    
5.  Dentro da Unity, navegue até a pasta `Assets/Scenes/Home` e abra a cena principal do jogo.
    
6.  Clique no botão **Play** (▶) no topo do editor para testar.
